# Шаг 6 — Список дел

![Скриншот](images/step6.png)

Панель **«Список дел»** внизу окна собирает пометки `TODO, FIXME, XXX` и другие из исходного кода.

Вы увидите список всех найденных меток с указанием файла и строки. Клик по элементу открывает файл в редакторе. **Фильтры** позволяют сузить поиск: по тегу (`TODO, FIXME`) или по области (весь проект, конфигурация, расширения, эпики, фичи). **Группировка** — по файлу или иерархически по области. В настройках вы можете задать свои теги и шаблоны включаемых/исключаемых файлов.
