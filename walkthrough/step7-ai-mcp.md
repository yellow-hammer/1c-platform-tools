# Шаг 7 — Интеграция с AI и MCP

![Скриншот](images/placeholder.svg)

Расширение помогает AI-агентам (Cursor, Claude, Copilot) работать с проектом 1С: агент узнаёт, какие команды вызывать, и может выполнять их через навык или MCP.

**Навык для агента.** В группе «Навыки для AI» выполните команду **«Добавить»** и выберите, куда установить (для Cursor — «Для Cursor»). В проекте появится файл с инструкциями (SKILL.md): агент будет предлагать выполнять команды расширения (загрузка конфигурации, запуск Конфигуратора и т.д.) вместо ручного запуска скриптов.

**Запуск по файлу.** Если агент умеет создавать файлы, он может записать в `.cursor/1c-platform-tools-run-command` одну строку — идентификатор команды (например `1c-platform-tools.run.designer`). Расширение выполнит команду и удалит файл.

**MCP-сервер.** Для Cursor и других сред с поддержкой MCP в расширении есть MCP-сервер (инструменты `run_designer`, `run_enterprise`, `load_configuration`, `dump_configuration`, `run_command`). Настройка: откройте файл mcp.json в настройках Cursor и добавьте сервер `1c-platform-tools`, указав путь к `resources/mcp-server/index.mjs` в папке расширения и аргумент `${workspaceFolder}`. Подробности — в основном README расширения, раздел «Интеграция с AI и MCP».
